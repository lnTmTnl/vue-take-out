# 基于Vue3开发的外卖平台。
技术栈：Vue 3 + Vite + Vue Router + axios + Element Plus。（前端）

2022.10.15-下一步计划：引入Typescript替换JavaScript.

2022.10.5-当前进度：完成基本数据展示、用户登录、下单、查询订单的基本业务功能。
